What is Simple Inquest

Simple Inquest is, as the name implies, a simple to use enterprice search. The primary objective of the Project is to build a reliable tool for the inquest in the intern document pool.

Simple in terms of

  • Ease of use
  • Easy to set up

Features

The current version supports the folowing document types

Document types

  • MS Word, Excel and PowerPowerpoint
  • Plain text

What is Simple Inquest

Simple Inquest is, as the name implies, a simple to use enterprice search. The primary objective of the Project is to build a reliable tool for the inquest in the internal document pool.

Simple in terms of

  • Ease of use
  • Easy to set up
  • And easy to extend

Features

Simple Inquest use Apache Tika to extracting metadata and text content for various document formats. So the list of supported document formats is identical but not all are enabled by default.

Supported document formats

  • Microsoft Excel (application/vnd.ms-excel) (default)
  • Microsoft Word (application/msword) (default)
  • Microsoft PowerPoint (application/vnd.ms-powerpoint) (default)
  • Microsoft Visio (application/vnd.visio)
  • Microsoft Outlook (application/vnd.ms-outlook)
  • gzip compression (application/x-gzip)
  • bzip2 compression (application/x-bzip)
  • Extensible Markup Language (application/xml)
  • HyperText Markup Language (text/html)
  • Images (image/*)
  • Java class files
  • Java jar archives
  • MP3 Audio (audio/mp3)
  • OpenDocument (application/vnd.oasis.opendocument.*) (default)
  • Plain text (text/plain) (default)
  • Portable Document Format (application/pdf) (default)
  • Rich Text Format (application/rtf)
  • tar archive (application/x-tar)
  • ZIP archive (application/zip)
